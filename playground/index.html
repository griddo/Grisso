Customización (grisso.config.js)

Las clases de abajo se generan extendiendo la config por defecto vía grisso.config.js.

playground/grisso.config.js
module.exports = {
  extend: {
    backgroundColors: { brand: "var(--brand-1)" },
    shadows: { "2xl": "0 25px 50px -12px rgb(0 0 0 / 0.25)" },
    foregroundColors: { brand: "var(--brand-1)" },
  },
};

CSS Modules (tree-shaking con composes)

Las clases de abajo se preservan porque example.module.css las usa vía composes: ... from global.

playground/example.module.css
.card {
  composes: flex flex-col gap-sm p-md bg-ui shadow-lg from global;
}
.cardTitle {
  composes: text-1 font-bold from global;
}
.badge {
  composes: inline-flex items-center p-xs bg-3 text-3 from global;
}
